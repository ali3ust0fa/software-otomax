Format Pendukung Transaksi IP

Guna menunjang transaksi IP di versi 3.6.9 dihadirkan format - format baru terkait, sehingga dengan format tersebut diharapkan dapat mengurangi beban kinerja dari operator/penjaga server serta memudahkan bagi Sedoter (Pengambil Stok).

Format baru tersebut yakni:

  • UbahPasswordIP
  • UbahAlamatIP
  • UbahReportURL

Dengan format di atas Sedoter dapat mengubah Password IP, IP Address dan URL Report secara mandiri (tanpa bantuan operator/penjaga server). Adapun untuk detail formatnya dapat dicek di menu Setting -> Format Request (hanya tersedia di edisi yang mendukung transaksi IP klik disini).

Kemudian bila Anda belum mengerti tentang transaksi IP silahkan klik disini, klik disini, klik disini, klik disini dan klik disini. Silahkan dibaca semua kata per kata, halaman per halaman serta dipraktekkan. Dan setiap menjumpai kata "klik disini" silahkan diklik dan dibaca semua isinya.

Jika memiliki pertanyaan terkait fitur di atas silahkan kirimkan ke alamat email support@otomax-software.com atau klik disini
