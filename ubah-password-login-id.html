Ubah Password Login ID

Login ID dilengkapi dengan password. Anda dapat mengubah password tersebut secara periodik untuk meningkatkan keamanan. Adapun langkahnya sebagai berikut:

  1. Klik menu File -> klik Ubah Sandi...



  2. Akan tampil jendela di bawah



  3. Kemudian lakukan seperti gambar berikut



  4. Selanjutnya akan keluar jendela memberikan informasi bahwa pengubahan Password telah berhasil



  5. Klik OK, selesai.
