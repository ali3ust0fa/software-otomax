Harga Dinamis Sesuai Zona

Sama halnya dengan Pulsa Reguler klik disini, harga Pulsa Internet Bulk (Paket Data) di DigiPOS Outlet juga bermacam - macam sesuai lokasi nomor pelanggan sedang berada di zona mana, paket sama, lokasi nomor beda zona, beda harga. Di OtomaX, Anda dapat menjual Pulsa Internet Bulk Telkomsel dengan harga dinamis sesuai zona.

Tata caranya sama dengan menjual Pulsa Reguler dengan harga dinamis klik disini, yaitu:

  1. Anda harus punya OtomaX edisi Ultimate; kalau belum punya, bisa beli dahulu klik disini; apabila sudah punya tapi bukan edisi tersebut bisa upgrade dahulu klik disini; dan jika ingin membayar iuran tahunan OtomaX klik disini.
  2. Buat Kode Produk di Administrasi -> Produk seperti di atas, dan:
    1. Centang checkbox Postpaid.
    2. Pastikan kolom Hrg Beli isi: 0.
    3. Di kolom Hrg Jual isi jumlah laba yang Anda inginkan, contoh isi: 500.
  3. Buat kelompok jawaban di Administrasi -> Jawaban dari Provider seperti di atas, dan:
    1. Di baris penangkap jawaban dengan Kata Kunci Harus Ada: Silakan Masukkan PIN :, Prioritas: 0, ubah menjadi seperti di bawah ini:

      Kelompok Prioritas Kata Kunci Harus Ada Kata Kunci Tidak Boleh Ada Awalan Tujuan Akhiran Tujuan Awalan SN/Ref Akhiran SN/Ref Status Trx Expresi Reguler Perintah Parameter Balasan Catatan
      DIPO-NEL 0 Silakan Masukkan PIN : Menunggu Jawaban belian (?<paket>.+) dengan.*harga Rp. (?<tagihan>[,.\d]+) ke.*mor (?<tujuan>\d+) [pin] jawaban=Pembelian [paket] seharga Rp. [tagihan] ke [tujuan] sedang diproses
      DIPO-NEL 0
      DIPO-NEL 0
      DIPO-NEL 0

    2. Hapus penangkap jawaban dengan Kata Kunci Harus Ada: Silakan Masukkan PIN :, Prioritas: 1.
  4. Buat modul Dial seperti di atas.
  5. Buat parsing di Parsing ke Provider seperti di atas.
  6. Selesai.
