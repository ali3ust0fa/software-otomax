Jawaban Per Provider

Fitur ini berguna untuk memberikan balasan berbeda - beda untuk setiap Provider (Telkomsel, XL, Indosat, dan lain - lain) kepada Reseller. Perlu diketahui fitur ini hanya tersedia di Edisi Ultimate (lihat edisi - edisi OtomaX klik disini), sehingga jika Anda ingin mendapatkan fitur ini tiada jalan lain kecuali dengan melakukan upgrade OtomaX, klik disini untuk mengetahui tata cara upgrade.

Pemanfaatan fitur ini terbilang sangat sederhana, namun tak jarang yang kebingungan saat mensettingnya. Berikut langkah - langkahnya:

  1. Masuk menu Administrasi -> Provider, arahkan pointer mouse ke judul tabel Provider -> klik kanan -> pilih -> Format Sukses; atau (khusus v4.0.0 atau lebih tinggi) arahkan kursor ke judul tabel Provider -> klik kanan -> pilih Select columns... -> centang checkbox: Format Sukses -> klik OK.

  2. Akan muncul kolom baru bernama Format Sukses, masukkan balasan yang diinginkan pada kolom tersebut, misalnya untuk Provider Telkomsel diisi:

    Pembelian 1 buah voucher simPATI senilai <[nominal]> untuk <[tujuan]> telah berhasil. SN <[sn]>. Saldo (Rp[saldolama]-Rp[harga])=Rp[saldo]

    Untuk Provider XL diisi:

    Transaksi dgn No. [sn] berhasil utk pengisian pulsa ke [tujuan] sebesar ([nominal]000). Saldo (Rp[saldolama]-Rp[harga])=Rp[saldo]

    Untuk Provider Indosat diisi:

    Isi Rp.[nominal]000 Regular pd [waktu] ke [tujuan] SUKSES.Ref:[sn]. Saldo (Rp[saldolama]-Rp[harga])=Rp[saldo]

    Terakhir klik tombol Simpan.

  3. Selesai.

Dengan disetting seperti di atas setiap transaksi sukses balasan ke Reseller akan mengacu pada apa yang ada di kolom Format Sukses.
