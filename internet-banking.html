Internet Banking

Modul Internet Banking berguna untuk membaca mutasi transaksi bank BCA, BNI, BRI, Mandiri (Internet), Mandiri Online, dan Mandiri Bisnis sehingga deposit Reseller dapat masuk secara otomatis (tanpa campur tangan operator server) sesaat setelah Reseller melakukan transfer. Modul ini hanya tersedia di OtomaX edisi Standard, Pro, Advanced, Enterprise dan Ultimate klik disini. Apabila Anda belum mempunyai OtomaX klik disini; dan apabila sudah memiliki namun belum edisi tersebut untuk menikmatinya Anda harus melakukan Upgrade klik disini; dan apabila ingin membayar iuran tahunan OtomaX klik disini.

Agar dapat bekerja, modul Internet Banking di OtomaX membutuhkan beberapa software pendukung, yaitu Internet Explorer versi 11, Transport Layer Security (TLS) v1.2 (khusus windows 7) untuk modul BNI, BRI dan Mandiri, serta Google Chrome, Chrome Driver, Selenium WebDriver untuk modul Mandiri Online dan Mandiri Bisnis. Adapun modul BCA bisa bekerja tanpa software pendukung. Oleh karena itu, pastikan Anda telah menginstallnya sebelum mensetting modul Internet Banking OtomaX klik disini.

Catatan: Di OtomaX edisi Standard dan Pro terdapat modul Internet Banking mulai versi 4.0.0.

BCA

Berikut ini langkah - langkah mensetting modul BCA:

  1. Pastikan Anda sudah mempunyai akun Internet Banking BCA, kalau belum silahkan datang ke kantor Bank BCA terdekat dari tempat tinggal Anda.
  2. Pastikan pengaturan bahasa di akun internet banking BCA Anda telah menggunakan bahasa indonesia. Langkahnya:
    1. Login internet banking BCA via web.
    2. Klik menu Administrasi -> lalu klik Ubah Bahasa.
    3. Pada pilihan bahasa pilih Indonesia dan kemudian klik tombol OK.
    4. Logout.
    5. Selesai.
  3. Pastikan Edisi OtomaX Anda Standard, Pro, Advanced, Enterprise atau Ultimate sebab modul ini hanya tersedia di edisi tersebut.
    1. Untuk OtomaX v3.9.1 atau lebih rendah, pilih modul BCA -> klik kanan -> klik Setting -> kemudian lakukan sesuai petunjuk di gambar di bawah ini:



    2. Untuk OtomaX v4.0.0 atau lebih tinggi, pilih modul Internet Banking -> klik kanan -> klik Tambah Internet Banking... -> kemudian lakukan sesuai petunjuk di gambar di bawah ini:

  5. Selesai
