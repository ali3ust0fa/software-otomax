Variabel HTTP

Dari sekian banyak fitur baru di v4.0.0 adalagi fitur bernama Variabel HTTP GET/POST. Apasih gunanya?... dengannya Anda dapat menciptakan variabel baru beserta nilainya yang dapat diakses di parsing. Fitur ini hanya berlaku untuk modul IP dan hanya tersedia di OtomaX edisi Ultimate. Namun tak perlu khawatir jika OtomaX Anda bukan edisi Ultimate, sebab Anda dapat melakukan upgrade klik disini; apabila belum mempunyai OtomaX penasaran ingin mencoba dapat membeli dahulu klik disini.

Penggunaan fitur ini cukup mudah dan dapat diilustrasikan sebagai berikut:

  1. Buat nama variabel beserta nilainya, misalnya nama variabel: secretkey dan nilainya: 004uXs0.
  2. Masuk menu Administrasi -> Parsing ke Provider -> pada Modul pilih modul IP -> kemudian pada kolom Perintah tambahkan &secretkey=004uXs0

    trx?memberID=[memberid]&product=S[nominal]&dest=[tujuan]&secretkey=004uXs0

    Variabel secretkey kini dapat Anda ambil nilainya dengan menambahkan keyword: var_ di depan nama variabel sehingga menjadi [var_secretkey].

    trx?memberID=[memberid]&product=S[nominal]&dest=[tujuan]&secretkey=004uXs0&key=[var_secretkey]

    Data yang dikirimkan ke supplier akan melampirkan nilai variabel secretkey yaitu 004uXs0 sebagai berikut:

    trx?memberID=SVR3478&product=S5&dest=081234567890&key=004uXs0

  3. Selesai.

Anda dapat menggunakan fitur ini untuk beberapa kasus dimana syntax supplier membutuhkan variabel tambahan selain yang sudah umum di OtomaX seperti memberid, product, dest, refid, sign. Dengan ini variabel parsing OtomaX semakin flexible dan dapat disesuaikan sesuai kebutuhan.
