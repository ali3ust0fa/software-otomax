Perubahan OID (3.7.1)

OID singkatan dari OtomaX ID, sejumlah kode unik yang menjadi identitas masing - masing software OtomaX. Dewasa ini, OID berkembang menjadi sebuah komponen untuk Host to Host ke sesama software OtomaX dijembadani oleh modul OtomaX HTH. Oleh karenanya, perubahan OID baik di sisi Supplier maupun Sedoter akan mempengaruhi bekerja dan tidaknya transaksi melalui OtomaX HTH.

Pada versi 3.7.1, OID mendapat pembaharuan yang berdampak pada perubahan OID OtomaX Anda. Hal ini akan terjadi ketika Anda melakukan update OtomaX dari versi di bawah 3.7.1 ke versi 3.7.1 atau lebih tinggi. Akibatnya modul OtomaX HTH Anda akan berwarna kuning dan transaksi melalui OtomaX HTH tidak jalan. Tak perlu panik karena itu merupakan hal normal, cukup lakukan beberapa hal berikut:

  1. Pilih modul OtomaX HTH -> klik kanan -> klik Setting.
  2. Pada box: Password masukkan password OtomaX HTH Anda. Apabila Anda lupa password, lakukan reset password klik disini.
  3. Klik tombol Simpan. Sampai langkah ini modul OtomaX HTH Anda berwarna biru, tanda sudah bekerja kembali.
  4. Dari Nomor HP atau Jabber Anda yang terdaftar di Supplier, ketik: UBAHOID OID_BARU_ANDA PIN (atau sesuaikan format ubah OID di server Supplier Anda) kirim ke SMS Center atau IM Center Supplier Anda (ganti kata: OID_BARU_ANDA dengan OID OtomaX Anda yang baru).
  5. Selesai.

Selain berpengaruh pada OtomaX HTH, perubahan OID di versi 3.7.1 akan berpengaruh juga pada modul IP Gateway. Modul tersebut akan tidak bekerja, lakukan hal berikut agar bekerja kembali:

  1. Pilih modul IP Gateway -> klik kanan -> klik Setting.
  2. Pada box: Password masukkan password IP Gateway Anda.
  3. Klik tombol Save.
  4. Selesai.

Anda mungkin akan menjumpai pesan " OID saat ini tidak cocok dengan yang tersimpan di database ". Apabila Anda menjumpainya cukup klik disini dan ikuti instruksi yang ada di halaman tersebut langkah demi langkah.

Imbas dari perubahan OID, OtomaX Anda mungkin tidak dapat berjalan sempurna dengan pesan " ArgumentNullException: Value cannot be null. Parameter name: disk " atau " ArgumentNullException: Value cannot be null. Parameter name: sys "; yang perlu Anda lakukan adalah melakukan update ke versi 3.7.2 atau lebih tinggi klik disini.

Itulah beberapa pengaruh yang ditimbulkan oleh perubahan OID di versi 3.7.1, setidaknya terdapat 4 hal sebagaimana termaktub di atas; dua teratas terjadi di semua user, dan dua terakhir hanya terjadi di sebagian user.
