Web SMS Sender (3.8.4)

Web SMS Sender telah tersedia mulai versi 3.8.3 klik disini, namun hanya support di sebagian penyedia SMS Gateway disebabkan masalah parameter nomor tujuan. Guna menanggulanginya dan agar lebih flexible mendukung lebih banyak penyedia SMS Gateway, di versi 3.8.4 ditambahkan beberapa parameter.

Sebelum membahas tentang parameter, perlu Anda ketahui bahwa fitur ini hanya tersedia di OtomaX edisi Ultimate klik disini; jadi jika menginginkannya dan edisi OtomaX Anda belum Ultimate silahkan upgrade dulu klik disini; dan jika belum memiliki OtomaX silahkan membeli dulu klik disini; dan jika ingin membayar iuran OtomaX klik disini.

Parameter - parameter yang ditambahkan dan dapat digunakan di Web SMS Sender di versi 3.8.4 adalah sebagai berikut:

  1. [to]
    Untuk mengirimkan nomor tujuan ke penyedia SMS Gateway berupa nomor hp lengkap dengan tanda + dan kode negara, contoh: +6281234567890. Gunakan ini apabila penyedia mensyaratkan nomor tujuan melampirkan tanda + dan diikuti kode negara.

  2. [to2]
    Untuk mengirimkan nomor tujuan ke penyedia SMS Gateway berupa nomor hp tanpa tanda + dan dengan kode negara, contoh: 6281234567890. Gunakan ini apabila penyedia mensyaratkan nomor tujuan melampirkan kode negara tanpa diikuti tanda +.

  3. [to3]
    Untuk mengirimkan nomor tujuan ke penyedia SMS Gateway berupa nomor hp terpotong tanpa tanda + dan tanpa kode negara, contoh: 81234567890. Gunakan ini apabila penyedia mensyaratkan nomor tujuan tidak melampirkan tanda + dan kode negara, misalnya penyedia mensyaratkan nomor tujuan diawali dengan 0 tanpa tanda + dan kode negara, contoh: 081234567890, maka pada Web URL isi: 0[t03].

Kemudian apabila Anda mempunyai pertanyaan seputar fitur diatas klik disini.
