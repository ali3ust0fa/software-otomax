Markup Produk

Di tengah makin banyaknya produk yang diusung, dijual oleh server pulsa dari semula hanya pulsa handphone menjadi bermacam - macam produk, sehingga diperlukan sistem dinamis, diantaranya tentang markup harga; misalnya: produk dengan harga modal Rp500.000-an kurang senang rasanya bagi sebagian upline jika komisi yang didapatkan hanya Rp100, sama dengan transaksi isi ulang pulsa dengan harga modal Rp5.000-an sebagaimana terjadi di versi sebelumnya; produk apapun, modal berapapun, hanya dapat komisi Rp100, tidak dapat dibeda - bedakan. Guna mengkomodasi hal tersebut, di versi 3.7.6 OtomaX menghadirkan fitur baru yaitu Markup Produk. Dengannya upline dapat menentukan markup berbeda - beda untuk setiap produk; misalnya: produk dengan modal Rp500.000-an dimarkup Rp500, produk dengan modal Rp5.000-an dimarkup Rp100; dan seterusnya. Ini tentu akan meningkatkan semangat dan daya saing upline dalam menggaet calon downline yang bermuara pada meningkatnya transaksi di server pulsa Anda.

Perlu diketahui fitur ini hanya tersedia di edisi Standard atau lebih tinggi klik disini, oleh karena itu jika Anda membutuhkannya dan edisi OtomaX Anda belum edisi tersebut lakukan upgrade dahulu klik disini; dan jika Anda belum memiliki OtomaX, beli dahulu klik disini; dan jika versi OtomaX Anda belum 3.7.6, update dahulu klik disini; dan jika masa berlaku OtomaX Anda telah berakhir, bayar iuran dahulu klik disini.

Markup Produk dapat dilakukan dari server maupun oleh upline via SMS atau Chat secara mandiri dengan format tertentu yang akan dibahas di bawah ini. Namun sebelum menyimaknya sebaiknya Anda klik disini terlebih dahulu:

  1. Klik Administrasi -> Markup Produk:



    Di menu di atas Anda dapat mengecek, menentukan ataupun mengubah besaran markup untuk setiap produk.

  2. Klik Setting -> Format Request -> Key: UbahMarkupProduk:



    Upline dapat mengubah besaran markup tiap produk dengan format di atas, contoh: MP.OX0003.S5=50,S10=75,S20=100.1234 atau MP2.PLN20=100,PLN50=150,PLN100=200.1234.

  3. Klik Setting -> Format Request Key: CekMarkupProduk:



    Upline dapat mengecek markup produk dengan format di atas, contoh: CMP.OX0003.S5.1234 atau CMP.OX0003.1234.

  4. Klik Setting -> Format Request -> Key: HapusMarkupProduk:



    Upline dapat menghapus markup produk dengan format di atas, contoh: DELMP.OX0003.S5,S10,S20.1234, DELMP2.OX0003.1234 atau DELMP3.S50.1234.

  5. Selesai

Kemudian, bila Anda mempunyai pertanyaan tehnis seputar Markup Produk, silahkan kirimkan melalui email ke alamat: support@otomax-software.com atau klik disini.

Catatan 1: Markup Produk belum memiliki setting untuk membatasi batasan maksimal markup per produk, adapun MaxMarkup di Setting -> Lain - lain key: MaxMarkup berlaku untuk Markup (Reseller), belum berlaku untuk Markup Produk.

Catatan 2: mulai versi 3.7.8 MaxMarkup di Setting -> Lain - lain key: MaxMarkup berlaku juga untuk Markup Produk (Reseller) klik disini.
