Menghapus data

Seiring berjalannya transaksi, database OtomaX akan mengalami penambahan jumlah. Agar OtomaX tetap dalam performa prima sebaiknya dilakukan pemotongan data dari database secara berkala, 1 minggu sekali, 2 minggu sekali, 1 bulan sekali tergantung besar kecilnya transaksi, sebab semakin banyak transaksi semakin besar pula datanya.

Berikut akan dijelaskan tata menghapus data:

  1. Masuk menu Database -> Menghapus Data...



  2. Akan muncul jendela sebagai berikut:



  3. Tentukan tanggal data yang akan dihapus, lalu tentukan data yang ingin dihapus, pada contoh ini misalnya semua Data Inbox, semua Data Outbox, Data Transaksi. Kemudian centang Shrink Database. Terakhir klik Tombol Hapus untuk melakukan proses penghapusan data.



  4. Selesai.
