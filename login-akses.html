Login Akses

Di versi 3.5.0 hadir fitur baru yakni Login Akses, berguna untuk menentukan hak akses pemakai. Login Akses memiliki 2 jenis Login ID: Default dan Non Default.



Default Login Akses
Login ID : admin
Password : admin

Default merupakan Login ID bawaan OtomaX, tidak dapat diubah dan tidak pula dihapus, memiliki akses penuh ke semua menu OtomaX. Login ID tersebut adalah admin, yang dibuat oleh sistem secara otomatis. Ini tersedia di OtomaX edisi free maupun berbayar.

Adapun Non Default merupakan Login ID yang dibuat oleh Owner server, dapat diubah, dapat dihapus, dan dapat ditentukan hak aksesnya pada menu - menu OtomaX. Ini hanya tersedia di minimal OtomaX edisi Standard atau lebih tinggi klik disini. Bila edisi OtomaX Anda belum mendukung, upgrade dahulu klik disini, dan bila belum memiliki OtomaX, beli dahulu klik disini. Serta bila ingin mencoba OtomaX Free Edition download disini.
