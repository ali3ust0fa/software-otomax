Setting Web Report

Web Report tentu membutuhkan URL atau alamat untuk mengaksesnya berupa subdomain atau domain, contohnya orisinil.report.web.id. Perlu Anda ketahui bahwa URL Web Report OtomaX dapat menggunakan subdomain dari domain yang telah disediakan OtomaX yakni report.web.id (gratis), dan dapat pula menggunakan domain milik Anda sendiri (yang telah Anda beli di penyedia domain).

Berikut ini langkah mensetting Web Report OtomaX:

  1. Pastikan telah mensetting modul Online Backup, kalau belum klik disini.
  2. Pastikan telah menggunakan modul Online Backup v1.23 atau lebih tinggi, kalau belum download disini, kemudian timpa file Online Backup lama dengan file yang baru Anda download.
  3. Pastikan layanan Web Report telah Anda aktivasi, kalau belum klik disini.
  4. Buka modul Online Backup -> Klik 3 tanda titik bertumpuk di sebelah kiri atas -> klik Restart Engine.
  5. Jika ingin menggunakan URL berupa subdomain dari domain yang telah disediakan OtomaX, maka lakukan hal berikut:
    1. Klik 3 tanda titik bertumpuk di sebelah kiri atas -> klik Settings -> klik tab Web Report, kemudian lakukan sesuai petunjuk di gambar bawah ini:



      Pada contoh di atas web report dapat diakses dengan mengunjungi URL serverpulsa.report.web.id.
    2. Selesai.
  6. Jika ingin menggunakan URL berupa domain Anda sendiri, maka lakukan hal berikut:
    1. Misalnya domain Anda adalah mydomain.com, maka pastikan Anda telah membelinya di penyedia atau penjual domain.
    2. Klik 3 tanda titik bertumpuk di sebelah kiri atas -> klik Settings -> klik tab Web Report, kemudian lakukan sesuai petunjuk di gambar bawah ini:



      Pada contoh di atas web report dapat diakses dengan mengunjungi URL mydomain.com.
    3. Hubungi penyedia atau penjual domain Anda, minta agar domain mydomain.com dipointing ke IP Akun Online Backup Anda.
    4. Selesai. Web report Anda akan dapat diakses jika proses propagansi domain di penyedia domain telah selesai, biasanya membutuhkan waktu sekitar 1x24 jam atau lebih tergantung penyedia domain.
  7. Selesai.

Apabila Anda ingin mengetahui demo Web Report OtomaX, silahkan klik disini.
