Mandiri Online (4.0.0)

Modul Mandiri Online merupakan salah satu dari modul Internet Banking OtomaX, telah tersedia sejak v3.7.5 dengan menggunakan software pendukung Internet Explorer, klik disini. Di v4.0.0 modul ini menggunakan software pendukung lain, yaitu Google Chrome, Chrome Driver dan Selenium Web Driver. Agar dapat bekerja, pastikan Anda telah menginstalnya klik disini untuk menginstal.

Perlu diketahui fitur ini hanya tersedia di edisi Standard, Pro, Advanced, Enterprise, dan Ultimate mulai v4.0.0 klik disini. Jika edisi OtomaX Anda bukan salah satu dari edisi tersebut lakukan upgrade, klik disini untuk upgrade; jika belum v4.0.0, klik disini untuk update; dan jika belum memiliki OtomaX, klik disini untuk membeli, stok melimpah....

Penggunannya cukup mudah, pilih modul Internet Banking -> klik kanan -> klik Tambah Internet Banking... -> beri nama modul dan seterusnya, untuk lebih lengkapnya silahkan klik disini.
