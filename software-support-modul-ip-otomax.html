Software Support Modul IP OtomaX

Di OtomaX terdapat modul yang dapat digunakan mengambil stok dari server lain menggunakan IP, namanya ialah Modul IP atau IP Client dengan beberapa syarat tehnis tergantung software yang digunakan supplier, sehingga Modul tersebut tidak serta merta dapat digunakan menembak via IP ke software pulsa apa saja, ada beberapa software pulsa yang mendukung, dan ada pula yang tidak. Pada kesempatan ini kami akan merangkum hal tersebut, namun sebelum membahas hal itu, Anda perlu mengetahui terlebih dahulu hal-hal yang harus dipenuhi agar dapat transaksi via IP, selengkapnya klik disini.

Berikut ini rangkuman software-software pulsa yang dapat ditembak dengan Modul IP atau IP Client OtomaX dan yang tidak:

Sekarang, jika Anda ingin host to host atau mengambil stok dari server lain via IP silahkan tanya ke supplier Anda: software apa yang mereka gunakan dan Anda cocokkan dengan tabel di atas, apakah software mereka mendukung atau tidak. Jika belum tercantum di tabel di atas silahkan minta dokumen spesifikasi (spek) transaksi IP yang dimiliki supplier Anda dan berikan ke kami untuk ditinjau.

Bagaimana kalau yang di tabel di atas dibalik, misalnya ST24 ke OtomaX, Flash Machine ke OtomaX dan seterusnya, apakah support? untuk mendapatkan jawaban perihal tersebut silahkan hubungi team support software ST24, Flash Machine dan lainnya. Adapun spesifikasi (spek) transaksi IP OtomaX selengkapnya klik disini.

Walhasil, Modul IP atau IP Client OtomaX dapat digunakan host to host ke server lain atau tidak tergantung spesifikasi transaksi IP software yang digunakan oleh supplier. Perlu diketahui Modul IP mendukung metode XML, HTTPGET dan HTTPPOST.

Kemudian, bila Anda memiliki pertanyaan-pertanyaan seputar tehnis transaksi via IP di OtomaX silahkan kirimkan pertanyaan Anda ke alamat email support@otomax-software.com atau klik disini.
