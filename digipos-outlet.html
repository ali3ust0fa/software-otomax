DigiPOS Outlet

DigiPOS Outlet merupakan sistem milik Telkomsel untuk mengisi ulang Pulsa Regular, Pulsa Internet (Bulk), Kuota Nelpon, Produk Digital seperti Games, Paket SMS dan lain sebagainya berbasis Aplikasi Mobile, Web dan USSD (Dial) dengan 2 pilihan Metode Pembayaran, yaitu LINKAJA dan NGRS. Pada kesempatan ini yang akan dibahas adalah basis USSD produk Pulsa Regular dengan Metode Pembayaran LINKAJA.

Berikut ini langkah mengintegrasikan sistem DigiPOS Outlet basis USSD dengan OtomaX:

Transaksi Harus Dengan Kode

  1. Masuk menu Administrasi -> Produk -> buat Kode Produk seperti petunjuk di bawah ini:

  2. Masuk menu Administrasi -> Jawaban dari Provider -> buat penangkap jawaban seperti di bawah ini:

    Kelompok Prioritas Kata Kunci Harus Ada Kata Kunci Tidak Boleh Ada Awalan Tujuan Akhiran Tujuan Awalan SN/Ref Akhiran SN/Ref Status Trx Expresi Reguler Perintah Parameter Balasan Catatan
    DIPO 0 Masukkan nomor pelanggan Menunggu Jawaban [tujuan]
    DIPO 0 5. Paket Lainnya 4. Isi Pulsa Reguler Menunggu Jawaban 5
    DIPO 0 4. Isi Pulsa Reguler 5. Paket Lainnya Menunggu Jawaban 4
    DIPO 0 1. 5 K
    2. 10 K
    3. 20 K
    4. 25 K
    5. 50 K
    6. 100 K
    		Menunggu Jawaban [nominal]
    DIPO 0 Silakan Masukkan PIN : Menunggu Jawaban senilai Rp. (5.400|5.500|5.600|5.700|10.700|20.700|25.700|50.700|100.200) [pin] Harga Sesuai Keinginan > Diproses
    DIPO 1 Silakan Masukkan PIN : Menunggu Jawaban senilai Rp. (?<harga>\S+) ke [] jawaban=Tidak diproses, [harga] belum tercantum di Expresi Reguler Harga Tidak Sesuai Keinginan > Tidak Diproses
    DIPO 0 Terima kasih permintaan Anda sedang diproses Menunggu Jawaban
    DIPO 0 berhasil 62 <spasi>telah SN<spasi>:<spasi> <spasi>pada Sukses tujuan=0[tujuan]
    DIPO 0
    DIPO 0
    DIPO 0

    Keterangan:

    1. Ganti <spasi> dengan menekan tombol spasi di keyboard 1 kali .
    2. Ganti 5.400,5.500,5.600,5.700,10.700,20.700,25.700,50.700,100.200 dengan harga yang Anda inginkan.
    3. Penangkap jawaban ini untuk memproses transaksi denom 5, 10, 20, 25, 50 dan 100.

  3. Pilih Dial -> klik kanan -> klik Tambah Dial... -> klik Page 1 -> di Label misal isi: DIPO-1; Terminal pilih Terminal berisi chip DigiPOS Outlet; Kelompok Jawaban pilih kelompok DIPO; Parameter Parsing isi: pin=1234 (ganti 1234 dengan pin chip Anda); ID/Nomor Sender/Pengirim Provider isi: TELKOMSEL.

    Klik Page 2 -> di Default Parsing/Command isi: *181*2*1#.

  4. Klik Administrasi -> klik Parsing ke Provider -> di Modul pilih DIPO-1 -> Produk isi misal: DT5 -> di kolom Perintah kosongkan; terakhir klik tombol Simpan. Perhatikan gambar di bawah ini:

  5. Selesai.

Catatan: Dengan setting diatas Reseller harus bertransaksi menggunakan Kode Produk, contoh: DT5.081234567800.1234 (TIDAK BISA TANPA KODE).
