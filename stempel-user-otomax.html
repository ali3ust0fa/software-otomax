Tanya:

Jawab:

  1. Anda harus memiliki Software Pulsa OtomaX berbayar, bila belum silahkan membeli terlebih dahulu, selengkapnya klik disini.
  2. Kirim email ke alamat email support@otomax-software.com, sampaikan Anda mengajukan permohonan stempel User OtomaX, lampirkan data - data berikut ini:
    • Username Forum OtomaX Anda.
    • IMSI / OID OtomaX.
    • Nama pemilik.
    • Alamat lengkap.
    • Serial Number (SN).
