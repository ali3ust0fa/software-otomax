Night Mode

Mengoperasikan server pulsa tak lepas dari namanya layar monitor, tingkat kecerahan tinggi pada layar akan mempengaruhi kejenuhan mata. Agar ramah dengan mata pengguna mulai versi 3.7.7 OtomaX menyajikan fitur baru bernama Night Mode klik disini. Sesuai namanya, jika fitur tersebut diaktifkan maka tampilan GUI OtomaX akan gelap seperti gambar di bawah ini:



Untuk mengaktifkan dan menonaktifkannya masuk menu File -> klik Night Mode, segera setelahnya tampilan OtomaX Anda akan gelap seperti malam hari dan mata Anda menjadi teduh, tidak mudah lelah atau merah bila berlama - lama di depan layar monitor. Selain itu karena redup konsumsi listrik relatif lebih sedikit, cocok untuk server yang nyala 24 jam.

Tertarik dengan fitur Night Mode?... segera update ke versi 3.7.7 atau lebih tinggi klik disini; dan jika belum mempunyai OtomaX klik disini; dan jika masa berlaku OtomaX Anda telah berakhir klik disini.
