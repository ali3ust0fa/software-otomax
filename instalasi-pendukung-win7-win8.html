Instalasi Software Pendukung

Windows 7 atau 8

Langkah menginstall Software Pendukung OtomaX untuk Windows 7 atau Windows 8 adalah sebagai berikut:

  1. Install Microsoft .Net Framework
  2. Install Internet Explorer versi 11
  3. Install Transport Layer Security (TLS) v1.2
  4. Install Google Chrome
  5. Install Chrome Driver
  6. Install Selenium WebDriver
  7. Install Microsoft SQL Server Express
  8. Install Microsoft SQL Server Management Studio Express

Install Microsoft .Net Framework

Pertama, install Microsoft .Net Framework sebagai berikut:

  • Cari file Microsoft .Net Framework yang sudah Anda download. Jika belum download silahkan download dahulu disini.
  • Klik 2x pada file Microsoft .Net Framework, akan muncul jendela seperti di bawah pilih I have read and accept the license terms. Kemudian klik Install.

  • Tunggu proses verifikasi data

  • Jika muncul popup seperti di bawah ini klik YES

  • Tunggu proses instalasi

  • Instalasi sukses, klik Finish.

Install Internet Explorer versi 11

Kedua, install Internet Explorer versi 11 sebagai berikut:

  1. Download Internet Explorer versi 11 klik disini.
  2. Cari file hasil download, klik 2x dan kemudian klik tombol Install.

  3. Tunggu proses instalasi.

  4. Instalasi selesai, klik tombol Restart now

  5. Setelah restart, buka Internet Explorer versi 11, tekan Alt + X pada keyboard, lalu pilih Internet Options.
    1. Untuk pengguna Windows 7 -> klik tab Programs, klik tombol Make Default -> klik tombol OK dan kemudian tutup Internet Explorer.
    2. Untuk pengguna Windows 10 -> Klik tab Programs -> klik Make Internet Explorer the default browser -> pada panel Programs (sebelah kiri) cari dan pilih Internet Explorer -> klik Set this program as default -> klik tombol OK -> klik tombol OK lagi dan kemudian tutup Internet Explorer.
  7. Selesai.

Catatan: Apabila Anda menggunakan Windows 7 pastikan telah Windows 7 SP 1 sebelum menginstall Internet Explorer versi 11; jikalau belum, download dan install Service Pack 1 (SP 1) terlebih dahulu klik disini.

Install Transport Layer Security (TLS) v1.2

Ketiga, install Transport Layer Security (TLS) v1.2 sebagai berikut:

  1. Download Transport Layer Security (TLS) v1.2 klik disini.
  2. Cari file hasil download, klik 2x, kemudian tunggu proses persiapan instalasi.

  3. Klik tombol Yes.

  4. Tunggu proses instalasi.

  5. Instalasi selesai, klik tombol Restart Now.

Install Google Chrome

Keempat, install Google Chrome sebagai berikut:

  1. Download Google Chrome, klik disini.
  2. Cari file hasil download, klik 2x, apabila muncul jendela seperti di bawah ini klik tombol Run.

  3. Apabila muncul jendela seperti di bawah ini klik tombol Yes.

  4. Tunggu proses persiapan instalasi.

  5. Tunggu proses instalasi.

  6. Masuk Local Disk (C): -> Program Files atau Program Files (x86) -> Google -> Update -> pada file GoogleUpdate ganti namanya menjadi GoogleUpdateNoUpdate.
  7. Selesai.

Install Chrome Driver

Kelima, install Chrome Driver sebagai berikut:

  1. Buka Google Chrome, pada search bar ketik: chrome://settings/help lalu tekan Enter di keyboard dan catat versi Google Chrome.

  2. Download Chrome Driver yang versinya sama dengan Google Chrome, klik disini. Misal Google Chrome-nya v78, maka download Chrome Driver v78.
  3. Cari file hasil download, extract dan kemudian copy serta paste file hasil extract di folder OtomaX lihat disini atau folder lokasi instalasi OtomaX Anda.
  4. Selesai.

Install Selenium WebDriver

Keenam, install Selenium WebDriver sebagai berikut:

  1. Download Selenium WebDriver, klik disini.
  2. Cari file hasil download, extract dan kemudian copy serta paste file hasil extract di folder OtomaX lihat disini atau folder lokasi instalasi OtomaX Anda.
  3. Selesai.
