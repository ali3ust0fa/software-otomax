Keterangan:

  • Harga.
    adalah sejumlah uang yang harus dibayarkan untuk mendapatkan OtomaX.
  • Terminal.
    adalah modul untuk menempatkan HandPhone atau Modem.
  • Tiket Deposit.
    adalah fitur penambahan deposit secara otomatis tanpa campur tangan operator server. Bank yang didukung: BCA, BNI, BRI dan Mandiri.
  • SMS Center.
    adalah modul untuk menerima request SMS transaksi dari reseller. Misalnya: cek saldo, transaksi isi ulang pulsa dan lain - lain.
  • SMS Sender.
    adalah modul untuk mengirimkan balasan SMS ke reseller akan suatu transaksi tertentu.
  • Dial.
    adalah modul untuk memproses transaksi yang menggunakan metode dial, misalnya: MKIOS, DOMPUL, MOBO.
  • Keypad.
    adalah modul untuk memproses transaksi yang menggunakan metode Keypad, misalnya: MTRONIK.
  • SMS.
    adalah modul untuk memproses transaksi yang menggunakan metode SMS, misalnya: SMARTFREN.
  • STK.
    adalah modul untuk memproses transaksi yang menggunakan metode STK, misalnya: ISEV.
  • IP Client.
    adalah modul untuk transaksi Host to Host ke server lain menggunakan IP.
  • OtomaX HTH.
    adalah modul untuk transaksi Host to Host ke sesama server OtomaX menggunakan OID OtomaX.
  • IM Center.
    adalah modul untuk menempatkan ID YM, ID Gtalk, ID Jabber/XMPP dan juga ID FB. Baik dipergunakan sebagai penerima pesan instant berisi transaksi ataupun mengirimkan pesan instant berisi transaksi dari atau ke Reseller.
  • IM Hybrid HTH.
    adalah modul untuk transaksi Host to Host ke server lain menggunakan YM, Gtalk atau Jabber/XMPP.
  • Jumlah Trx.
    adalah jumlah transaksi yang diperkenankan.
  • HLR.
    adalah fitur untuk menentukan nomor - nomor yang dapat diproses berdasarkan HLR.
  • Level/Grup.
    adalah fitur untuk mengatur sistem downline.
  • Harga Grup.
    adalah fitur untuk memberikan harga khusus untuk grup reseller tertentu.
  • Regular Expresi.
    adalah fitur penangkap jawaban dengan menggunakan sistem Regex.
  • Management Pengguna.
    adalah fitur untuk mengatur pengguna server dan hak aksesnya.
  • Update Harga Beli.
    adalah fitur yang berfungsi meng-update harga beli secara otomatis bilamana terjadi perubahan harga produk dari supplier Host to Host.
  • PPOB Prabayar.
    adalah transaksi Voucher Token PLN, Vocher TV Kabel, Voucher Games Online dan lain - lain.
  • PPOB Paskabayar.
    adalah transaksi pembayaran tagihan listrik, telpon, PDAM, tagihan cicilan kendaraan dan lain - lain.
  • Vtri Web.
    adalah transaksi isi ulang pulsa Three via web.
  • Jam Aktif & Limit.
    adalah fitur untuk mengatur jam aktif sebuah modul dan membatasi jumlah transaksi yang dapat diprosesnya.
  • Sistem Poin.
    adalah sistem untuk memberikan poin tertentu kepada grup reseller tertentu dari setiap transaksi sebagai dasar pemberian hadiah.
  • Jawaban Per Provider.
    adalah fitur untuk membuat balasan berbeda - beda ke reseller untuk setiap Provider.
  • Quantity.
    adalah fitur untuk menembak stok unit dari chip SD ke chip RS, misalnya SD MKIOS.
  • Iuran Tahunan.
    adalah sejumlah uang yang harus dibayarkan 1 tahun sekali. Untuk pembelian baru iuran akan dikenakan pada tahun kedua setelah pembelian (artinya gratis iuran 1 tahun pertama).
  • Balik Nama.
    adalah biaya pergantian nama pemilik software OtomaX dari pemilik lama ke pemilik baru, lazimnya terjadi pada software second ketika dijual. Untuk pembelian baru tidak dikenakan biaya ini.
  • Voucher Fisik.
    adalah fitur untuk menjual Voucher Fisik.
  • Indosat Hybrid.
    adalah fitur isi ulang pulsa indosat dengan Voucher Fisik Indosat dimana pulsa langsung masuk ke nomor pembeli (pembeli tidak perlu memasukkan kode voucher manual).
