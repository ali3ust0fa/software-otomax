Web Report

Guna memudahkan Reseller mengecek status transaksi, produk, downline dan lain sebagainya, terkadang server pulsa membutuhkan yang namanya Web Report, sebuah website berisi data - data transaksi Reseller dan yang terkait, dapat diakses melalui jaringan internet.

Kini OtomaX telah menghadirkan layanan Web Report tersebut, menyatu dengan layanan Online Backup. Artinya, jika Anda menginginkan Web Report, maka Anda harus terlebih dahulu mendaftar layanan Online Backup klik disini, baru kemudian aktivasi layanan Web Report; dan bagi yang sudah mendaftar layanan Online Backup tinggal melakukan aktivasi layanan Web Report. Adapun alur dan biaya aktivasi dijelaskan di bawah.

Untuk memiliki Web Report OtomaX Anda tidak perlu menyiapkan hosting ataupun domain/sub-domain karena sudah disediakan OtomaX. Pun begitu, bila Anda ingin menggunakan domain/sub-domain Anda sendiri untuk URL Web Report juga bisa.

Aktivasi

Biaya aktivasi layanan Web Report OtomaX besarannya mulai Rp500.000 tergantung sisa masa berlaku layanan Online Backup dan hanya dikenakan 1x saat aktivasi (untuk mengetahui jumlah pasti biaya aktivasi, kirimkan username akun Online Backup ke kami). Berikut ini alur aktivasinya:

  1. Pastikan Anda telah mendaftar layanan Online Backup, kalau belum silahkan mendaftar dahulu klik disini.
  2. Kontak Kami di ID Telegram: @mam_service atau HP/WA/TG: +628-5258-1112-30, sampaikan Anda ingin aktivasi layanan Web Report serta informasikan:
    1. Username, dan
    2. Email akun Online Backup Anda.
  3. Tunggu Kami memberikan jumlah uang yang harus Anda transfer beserta rekening tujuan transfer.
  4. Lakukan transfer sesuai jumlah dan ke rekening yang Kami berikan.
  5. Konfirmasikan bahwa Anda telah melakukan transfer.
  6. Setelah pembayaran Anda tervalidasi, aktivasi layanan Web Report akan Kami proses.
  7. Kami akan menginformasikan akun Web Report Anda berhasil diaktivasi.
  8. Selesai.
