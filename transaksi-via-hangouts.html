Transaksi via Hangouts

Selain via SMS, kini Reseller Anda dapat melakukan transaksi menggunakan Hangouts, sebuah layanan chat yang dibidani oleh Google. Untuk menikmati layanan tersebut Reseller harus membuat account gmail terlebih dahulu seperti disini dan minimal memiliki HP Android serta paket data internet tentunya. Begitu pula Anda (Owner Server) juga harus memiliki account gmail untuk dipasang di modul IM Center (untuk pemasangan sama dengan disini).

Di versi 3.7.0 terdapat fitur baru yang membantu Reseller agar dapat transaksi via Hangouts di OtomaX dengan mudah, sebuah format built-in untuk mengetahui ID Hangouts dan kemudian diparalelkan, info lengkap disini. Bila belum memakai versi 3.7.0 update dahulu disini dan bila belum memiliki OtomaX beli dahulu disini. Berikut langkah yang harus dilakukan oleh Reseller:

  1. Siapkan HP Android berikut paket data internet.
  2. Membuat account gmail disini.
  3. Hubungi Owner Server untuk meminta account gmail Server, misalnya: myserver@gmail.com.
  4. Buka HP Android -> Jalankan aplikasi Play Store.
  5. Pada box: search tulis: Hangouts dan kemudian install aplikasi Hangouts di HP Android.
  6. Jalankan aplikasi Hangouts dan masukkan account gmail yang telah dibuat beserta passwordnya.
  7. Klik icon + (di pojok kanan bawah) -> klik icon New conversion.
  8. Di kotak sebelah kiri icon seperti lensa ketik account gmail Server. Pada contoh ini ketik: myserver@gmail.com.
  9. Account myserver@gmail.com akan muncul, lalu klik untuk meng-invite.
  10. Setelah ter-invite, kirim pesan: myhangouts ke account gmail Server.



  11. Reseller akan mendapatkan ID Hangoutnya, misalnya: 3ease9r3u9jda9039000294rdsf@public.talk.google.com



  12. Paralelkan/tambahkan ID Hangouts di atas ke Server dengan format tambah Gtalk. Untuk format silahkan hubungi Owner Server.
  13. Selesai. Reseller sudah dapat transaksi via Hangouts.

Kemudian, bila Anda memiliki pertanyaan-pertanyaan tentang fitur di atas silahkan kirimkan pertanyaan Anda ke alamat email support@otomax-software.com atau klik disini.
