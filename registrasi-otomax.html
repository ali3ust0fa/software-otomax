Registrasi Software OtomaX

Setelah Anda melakukan pembelian Software OtomaX (disini) dan mendapatkan lisensi (SN/Serial Number) beserta file-Nya, Anda perlu meregistrasikan Software Anda sebagai berikut:

  1. Tutup OtomaX Free Edition terlebih dahulu.
  2. Extraxt file Software OtomaX _edisi.zip yang Kami berikan, dimana edisi berisi nama edisi OtomaX, misal: _standard.zip, _advanced.zip dan seterusnya.
  3. Copy dan paste ke folder OtomaX (lokasi yang disediakan untuk instalasi OtomaX, lihat disini); atau ke folder lokasi instalasi OtomaX Free Edition Anda. Bila muncul jendela konfirmasi: klik Copy and Replace.
  4. Jalankan OtomaX (bila muncul form login masukkan: Login ID: admin, Password: admin, kemudian klik tombol Login) -> masuk menu Bantuan -> Tentang -> Serial Number: Masukkan SN (Serial Number) yang Kami berikan -> Centang checkbox sebelah kiri tombol Simpan -> Klik tombol Simpan -> Klik OK -> Klik OK -> Tutup OtomaX -> Jalankan kembali OtomaX (bila muncul form login masukkan: Login ID: admin, Password: admin, kemudian klik tombol Login) -> akan muncul Form Registrasi: isi dengan data pribadi Anda -> Setelah terisi semua klik tombol Send/Kirim -> Klik OK -> Klik OK -> Tutup OtomaX -> Jalankan kembali.
  5. Registrasi selesai. Software OtomaX siap digunakan.

Catatan: Ketika melakukan registrasi pastikan Anda terhubung dengan internet.
