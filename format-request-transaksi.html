Format Request Transaksi

Format Request Transaksi letaknya di menu Setting -> Format Request -> key: Transaksi[1,2,3,dst], berguna untuk menentukan pesan Reseller yang diterima dianggap sebagai transaksi.

OtomaX memiliki keunikan tersendiri dalam hal format request transaksi dibanding software lain, dimana Pengguna dapat meng-kreasikan format request transaksi sesuai kebutuhannya, mau dibikin gimana aja bisa. Keleluasaan ini sangat berguna bagi Pengguna yang memiliki daya imaginasi, kreatifitas tinggi dan pemahaman kuat tentang alur kerja software OtomaX, pun begitu tidak jarang akan membuat Pengguna yang awam atau pengguna baru bingung bukan kepalang; dan kadang juga menimbulkan error. Oleh karena itu sebaiknya dibuat sedetail mungkin dan seramping mungkin agar tidak terjadi error diakibatkan bentrokan antar format request, seperti di bawah ini:

Uraian Format
Transaksi dengan kode dan pin [kodeproduk].[tujuan].[pin]
Transaksi ke-2 dengan kode dan pin [kodeproduk].[counter].[tujuan].[pin]
Transaksi multi dengan kode dan pin {[kodeproduk].[tujuan].}[pin]
Transaksi dengan kode dan tanpa pin [kodeproduk].[tujuan]
Transaksi tanpa kode dan dengan pin [nominal].[tujuan].[pin]
Transaksi ke-2 tanpa kode dan dengan pin [nominal].[counter].[tujuan].[pin]
Transaksi multi tanpa kode dan dengan pin {[nominal].[tujuan].}[pin]
Transaksi tanpa kode dan tanpa pin [nominal].[tujuan]
Transaksi dengan kode dan pin, serta SMS ke nomor pembeli [kodeproduk].[tujuan].[pin].[enduser]
Transaksi dengan kode dan tanpa pin, serta SMS ke nomor pembeli [kodeproduk].[tujuan].[enduser]
Transaksi tanpa kode dan dengan pin, serta SMS ke nomor pembeli [nominal].[tujuan].[pin].[enduser]
Transaksi tanpa kode dan tanpa pin, serta SMS ke nomor pembeli [nominal].[tujuan].[enduser]

Bagi yang masih awam kami sarankan untuk menggunakan format request transaksi standard seperti di atas dan hapus format request transaksi yang lain. Adapun bagi yang sudah expert silahkan dikreasikan sesuai keinginan dan kebutuhan masing - masing, berikut menangani kendala/error yang mungkin timbul.

Catatan: Format request SMS ke nomor pembeli hanya berlaku mulai edisi Standard atau lebih tinggi, info lengkap klik disini. Apabila edisi OtomaX Anda belum mendukung, upgrade dahulu klik disini.
