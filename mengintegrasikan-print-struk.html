Mengintegrasikan Print Struk

Di Aplikasi Android OtomaX telah tersedia fitur cetak struk berguna untuk mencetak struk transaksi ke kertas menggunakan printer klik disini. Akan tetapi apabila Anda mempunyai layanan Print Struk Web sendiri, Anda dapat mengintegrasikannya dengan Aplikasi Android OtomaX, sehingga Anda dapat leluasa membuat berbagai macam model struk sesuai keinginan Anda. Langkahnya sebagai berikut:

  1. Pastikan Anda telah memiliki Aplikasi Android OtomaX. Kalau belum, silahkan memesan dahulu klik disini.
  2. Pastikan Aplikasi Android OtomaX telah disetting dengan benar, klik disini.
  3. Buka OtomaX Online Backup -> klik 3 tanda titik bertumpuk di sisi kiri atas -> klik Manage APK.
  4. Pada jendela Manage APK klik Menu, lalu buat Menu baru dan pada kolom Targel Url isi dengan Url Print Struk Web milik Anda. Perhatikan gambar di bawah ini:

    Setelah itu klik tombol Save.
  5. Selesai.

    Sekarang buka Aplikasi Android OtomaX, Menu yang baru saja dibuat akan muncul dan ketika diklik akan menampilkan tampilan Print Struk Web Anda. Perlu diketahui bahwa menu baru terkadang membutuhkan waktu sinkronisasi data agak lama (tidak langsung muncul di APK) estimasi maksimal 1x48 jam.

Tertarik ingin mempraktekkannya namun belum mempunyai Aplikasi Android OtomaX, pesan sekarang juga sebelum ketinggalan klik disini.
