Tanya:

Bagaimana melakukan update OtomaX?

Jawab:

  1. Cek masa berlaku OtomaX Anda, masuk menu Bantuan -> Tentang -> box Masa Berlaku. Karena update hanya dapat dilakukan bila OtomaX dalam masa berlaku.
  2. Login ke Forum OtomaX dengan menggunakan account berstempel User OtomaX. Bila belum punya account klik disini dan klik disini.







  3. Pada halaman di bawah, download file update OtomaX sesuai edisi yang Anda miliki:



  4. Extraxt file update OtomaX hasil download dan kemudian tutup OtomaX.
  5. Copy dan paste ke folder OtomaX (lokasi yang disediakan untuk instalasi OtomaX, lihat disini); atau ke folder lokasi instalasi OtomaX Anda. Bila muncul jendela konfirmasi: klik Copy and Replace.
  6. Selesai.
