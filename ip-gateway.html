IP Gateway

Konektifitas merupakan hal vital dalam bertransaksi, ketika konektifitas terganggu transaksi otomatis akan terganggu, mula - mula muncullah IP Center klik disini guna mengoptimalkan fungsi IP Client klik disini. Namun dikarenakan IP Center mengharuskan pengguna mempunyai IP Publik Statis dimana biaya langganannya ke ISP (Internet Service Provider) relatif mahal, IP Center kurang dilirik oleh para pengguna sebagai konektifitas. Akhirnya OtomaX membuat sebuah terobosan yaitu IP Gateway, dimana dengannya para pengguna dapat transaksi IP tanpa harus memiliki IP Publik Statis sebagaimana IP Center. Ini menghantarkan OtomaX menjadi software pulsa yang pertama kali mendukung transaksi IP tanpa IP Publik Statis.

Perlu Anda ketahui IP Gateway mulai tersedia di versi 3.6.9, ia memiliki 2 fungsi:

  1. Menerima report/balasan status transaksi IP (IP Gateway - report). Fungsi ini berguna untuk melakukan host to host ke server lain via IP. Tersedia di semua edisi OtomaX Berbayar kecuali OtomaX kategori cabang klik disini. Oleh karena itu bila Anda pengguna software lain dan ingin transaksi IP tanpa IP Publik Statis silahkan membeli OtomaX, klik disini.
  2. Menerima transaksi via IP (IP Gateway - center). Fungsi ini berguna untuk menerima transaksi Sedoter via IP. Hanya tersedia di edisi Standard atau lebih tinggi klik disini, sehingga bila Anda ingin menjadi Supplier transaksi IP dan edisi OtomaX Anda bukan edisi tersebut silahkan upgrade dahulu, klik disini.
