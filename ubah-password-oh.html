Ubah Password OtomaX HTH

Password merupakan salah satu hal yang sensitif, demi alasan keamanan sebaiknya Password diubah secara berkala. Begitu pula dengan Password OH, ada baiknya untuk dirubah secara berkala.

Perubahan Password OH hanya dapat dilakukan bilamana telah membuat Password OH terlebih dahulu sebagaimana langkah - langkahnya telah dijelaskan disini. Adapun mekanisme pengubahan Password OtomaX HTH adalah sebagai berikut:

  1. Pilih Otomax HTH -> Klik Kanan -> Klik Setting





  2. Setelah Klik Submit tunggu proses pengubahan Password selesai





  3. Klik Setting -> Masukkan Password baru Anda di kotak Password



  4. Selesai

Jika memiliki pertanyaan terkait fitur di atas silahkan kirimkan ke alamat email support@otomax-software.com atau klik disini
