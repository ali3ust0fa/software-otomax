Kirim Uang

Selain berjualan pulsa dan paket data, di OtomaX Anda juga dapat menjual jasa kirim uang antar bank (BCA, BNI, BRI, Mandiri dan Bank-Bank lain), topup OVO, GOPAY, DANA, LinkAja dan layanan sejenis lainnya. Berikut ini langkah - langkah pengaturan di OtomaX:

  1. Cari Supplier klik disini.
  2. Pastikan telah menggunakan OtomaX edisi Ultimate; jika belum, silahkan upgrade dahulu klik disini.
  3. Pastikan telah menggunakan OtomaX v3.8.2 atau lebih tinggi; apabila belum silahkan update dahulu klik disini; dan apabila masa berlaku habis, silahkan membayar iuran dulu klik disini.
  4. Pilih menu Administrasi -> Provider -> buat Kode Provider.

  5. Pilih Administrasi -> Produk -> buat Kode Produk.

    • Menu Nominal harus diisi: 1 dan Rumus Harga pilih: 2. (Qty * Nominal) + HrgMarkup agar Reseller bebas menentukan nominal kirim/transfer uang atau topup e-wallet.
    • Menu Hrg Jual isi dengan keuntungan yang dihendaki.
    • Menu Qty yg diijinkan sesuaikan dengan minimal dan maksimal nominal yang didukung oleh supplier. 10000-1000000 artinya nominal minimal transaksi adalah Rp10.000; dan maksimal Rp1.000.000.
  6. Pilih Setting -> Format Request -> buat format request, misalnya: KIRIM.[kodeproduk].[tujuan].[qty].[pin].

    Reseller dapat melakukan transaksi dengan mengirimkan pesan: KIRIM.BCA.098775899.125001.1234. Ini artinya Reseller melakukan transfer uang ke nomor rek BCA 098775899 sebesar Rp125001.

  7. Buat modul untuk memproses transaksi (sesuaikan dengan metode transaksi yang didukung supplier). Misalnya supplier mendukung transaksi menggunakan jabber, maka buat modul IM Hybrid HTH.
  8. Pilih Administrasi -> Parsing ke Provider -> buat parsing (sesuaikan dengan format transaksi supplier). Misalnya formatnya adalah TRF.KODEPRODUK.TUJUAN.NOMINAL.PIN dan kode produk transfer uang ke BCA adalah BCA; maka di kolom Perintah isi dengan TRF.BCA.[tujuan].[qty].[pin].

  9. Pilih Administrasi -> Jawaban dari Provider -> buat penangkap jawaban klik disini.
  10. Pilih Administrasi -> Provider -> arahkan kursor ke judul tabel Provider -> klik kanan -> pilih Select columns... -> centang checkbox: Format Sukses -> klik OK -> kemudian masukkan format balasan sukses berikut ini pada kolom Format Sukses:

    Kirim uang ke nomor rek [kodeproduk] [tujuan] sebesar Rp[qty] SUKSES Ref: [sn]. Saldo (Rp[saldolama]-Rp[harga])=Rp[saldo]

  11. Selesai.
