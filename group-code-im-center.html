Group Code (IM Center)

Di modul IM Center mulai versi 3.7.7 terdapat hal baru, yakni sebuah box bernama Group Code of member allowed to use this center, berguna untuk menentukan Grup Reseller mana saja yang dapat mengirim pesan ke akun XMPP/Jabber Anda. Reseller yang Grup-nya tidak tercantum tidak dapat mengirim pesan kepadanya.

Berikut ini pengoperasiannya:

  1. Pilih modul IM Center -> klik kanan -> klik Setting, akan muncul jendela seperti di bawah ini:



  2. Pilih kode Grup Reseller yang ingin Anda beri akses dapat berkirim pesan ke akun XMPP/Jabber Anda.



  3. Masukkan kode Grup Reseller yang Anda pilih ke box: Group Code of member allowed to use this center, misalnya: H dan O (jika lebih dari 1 pisahkan dengan tanda koma), kemudian klik tombol Save.



  4. Selesai.

Selanjutnya apabila Anda memiliki pertanyaan seputar fitur di atas silahkan klik disini.
