VTri Web Service

VTri Web Service merupakan layanan berbasis web untuk melakukan transaksi isi ulang pulsa Tri yang disediakan oleh Mitra Operator Tri dengan kecepatan transaksi yang optimal. Kabar baiknya, layanan ini dapat diintegrasikan dengan OtomaX memanfaatkan modul IP Client.

Berikut ini langkah - langkah mengintegrasikannya:

  1. Daftarkan nomor VTri Anda di salah satu penyedia layanan VTri Web Service, diantaranya:
    1. Media Karya Nusantara (MKN), klik disini.

    2. Inova, klik disini.

    2. Isi data sesuai isian di website pendaftaran tersebut. Setelah Anda mengisi data dan klik tombol Submit, Anda akan mendapatkan User dan Password untuk transaksi di layanan VTri Web Service yang dikirimkan via SMS atau Email.

    PERHATIAN: Pada contoh ini layanan yang akan digunakan adalah VTri Web Service MKN. Jika menggunakan layanan dari yang lain sesuaikan dengan spesifikasi transaksinya.
