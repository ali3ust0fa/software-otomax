Filter Server (Exclusion)

Mulai versi 3.7.7 terdapat fitur baru di modul IM Center bernama Filter Server (Exclusion) berguna untuk menentukan domain - domain XMPP/Jabber mana saja yang tidak ingin dikirimi pesan oleh akun jabber Anda.

Misalnya akun jabber Anda: myaccount@jabber.com tidak ingin mengirim pesan ke otheraccount@xmpp.com, maka masukkan xmpp.com ke dalam box: Filter Server (exclusion), dan jika domain yang ingin dimasukkan lebih dari 1, pisahkan dengan tanda koma atau titik koma. Kemudian klik tombol Save. Perhatikan gambar di bawah ini:

Dengan setting demikian, akun myaccount@jabber.com tidak akan kirim pesan ke akun otheraccount@xmpp.com. Apabila memiliki pertanyaan silahkan klik disini; dan bila menginginkan XMPP/Jabber yang relatif stabil klik disini.
