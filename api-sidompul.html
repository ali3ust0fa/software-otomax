API SiDOMPUL

Mulai versi 4.1.2 kita dapat mentransaksikan produk - produk XL Axiata seperti isi pulsa dan paket data di OtomaX melalui API SiDOMPUL yang disediakan oleh XL Axiata.

Berikut langkahnya:

  1. Pastikan telah memiliki software OtomaX, untuk melakukan pembelian OtomaX silahkan klik disini.
  2. Pastikan Anda telah mendaftar API SiDOMPUL ke dealer XL Axiata di kota Anda dan memiliki data di bawah ini untuk mengaksesnya:
    1. PIN
    2. CLIENT_ID
    3. CLIENT_KEY
    4. API_ID dan API_KEY untuk API: GetDompulBalance
    5. API_ID dan API_KEY untuk API: GetProductList
    6. API_ID dan API_KEY untuk API: GetTransactionHistory
    7. API_ID dan API_KEY untuk API: PostEncrypt
    8. API_ID dan API_KEY untuk API: PostPackage
    9. API_ID dan API_KEY untuk API: PostW2P
  3. Pastikan telah menggunakan versi 4.1.2, kalau belum silahkan update dahulu klik disini.
  4. Di OtomaX, pilih modul API SiDOMPUL -> klik kanan -> klik Tambah API SiDOMPUL.

  5. Klik tombol Ubah API ID/KEY isi dengan data - data API SiDOMPUL Anda, kemudian klik tombol OK dan tombol Simpan.

  6. Pada menu Administrasi -> Parsing ke Provider -> kolom Perintah isi dengan: denom=NOMINAL atau paket=NAMA_PAKET.

  7. Buat penangkap jawaban, klik disini.
  8. Selesai.
